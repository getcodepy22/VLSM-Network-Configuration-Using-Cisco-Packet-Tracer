# VLSM-Network-Configuration-Using-Cisco-Packet-Tracer
Projet Cisco Packet Tracer basé sur le VLSM et le routage statique. Configuration de plusieurs routeurs, réseaux LAN/WAN, adressage IP optimisé, serveur et tests de connectivité. Simulation complète d’une architecture réseau utilisant le réseau 192.168.1.0/24.
